Skip navigation links

This inspection encourages a keyboard-accessible “skip to content” link near the top of the page so users can bypass repeated navigation.

Why this matters

Keyboard and screen reader users shouldn’t have to tab through large menus on every page. A visible‑on‑focus skip link provides a quick way to jump to the main content.

What’s checked

  • Presence of a skip link (e.g., <a class="skip-link" href="#main">).
  • Target exists and is descriptive (e.g., id="main", id="main-content").
  • Hidden links provide focus styles to become visible (e.g., :focus or :focus-visible).
  • Placement near top of <body>.

Quick‑fixes

  • (Planned) Insert a skip link after <body> when missing.
  • (Planned) Ensure or create a main content target id.
  • (Planned) Suggest a minimal CSS hint to reveal the link on focus.

Example

<style>.skip-link:focus{position:relative}</style>
<body>
  <a class="skip-link" href="#main-content">Skip to main content</a>
  ...
  <main id="main-content">...</main>
</body>
